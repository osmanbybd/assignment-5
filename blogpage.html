Question-1: What are the different ways to select an element in the DOM?

Selecting DOM elements is a fundamental aspect of web development with JavaScript. It allows developers to interact with and manipulate elements on a webpage dynamically. Proper selection of elements is crucial for tasks such as updating content, adding event listeners, or modifying styles.

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML retrieves the complete HTML content within an element, including tags, while innerText only returns the visible text content of an element, ignoring HTML tags and styling, and textContent returns all text content within an element, including hidden text like script tags, while also preserving whitespace and line breaks, unlike innerText which may adjust spacing based on rendering

Question-3: What is event delegation in the DOM?

event delegation is a JavaScript technique where you attach a single event listener to a parent element instead of adding listeners to multiple child elements,

Question-4: What is event bubbling in the DOM?

Event bubbling in JavaScript is a mechanism where an event triggered on a child element propagates upward through its ancestors in the DOM

Question-5: How do you create, add, and remove elements using JavaScript?

I can create new element in JS using document.createElement() and then add it to the DOM with appendChild() or insertBefore(). To remove an element, use removeChild() or parentNode.removeChild() on its parent element
